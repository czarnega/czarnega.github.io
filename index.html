Schemas

Top level collection of schemas. The information contained in the schemas provides enough information to build a smart client that knows what actions are available, what fields make up a resource, etc.

get

Retrieves a collection of Schema resources for each resource type.

get /schemas

Retrieves a collection of Schema resources for each resource type.
